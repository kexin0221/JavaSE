## `final` 断子绝孙

- `final`用在变量（或属性）时，这个变量成为常量，常量不允许再被改变（再次赋值）
- `final`用在方法时，标志这个方法到了"尽头"，可以被子代继承但不允许被重写
- `final`用在类时，标志这个类到了"尽头"，不允许再有子代
