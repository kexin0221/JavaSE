## Override 重写/覆盖

- **子类将父类的方法重新定义并实现一遍，称为方法重写（或覆盖）。**
- 方法重写的现实意义是，子类继承父类的行为主体，但是不沿用其行为，可以重新实现。 比如动物类都有捕食的方法，但是不同的动物：鱼、鹰、鸡各自捕食的方法有所不同。
- 方法重写是从**纵向**视角观察类的行为。
